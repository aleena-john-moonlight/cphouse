Network Charter v3.1

// 01. BANDWIDTH ALLOCATION

Access to the 1 Gbps Fiber Infrastructure is managed via dynamic percentage-based splitting. No single device is allocated the full 1 Gbps line.

"Total bandwidth is split among total active users by percentage. This ensures seamless entertainment and high-priority packet handling for all nodes simultaneously."

// 02. PROHIBITED ACTIONS & SECURITY

Zero Tolerance for Attacks

Attempts to perform DDoS, network sniffing, or unauthorized port-knocking on the C P House core will result in a permanent Hardware (MAC) Ban.

The guest network is fully isolated with enterprise-level firewall protection. Use of unauthorized VPN bypasses within the guest VLAN is strictly monitored.

// 03. PERMANENT NODE COMPLIANCE

The 48-Hour Rule

If you utilize the C P House network for a duration exceeding 2 consecutive days, you must register for a Static IP. This allows for optimized routing and prevents DHCP pool exhaustion.
